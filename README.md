# OpenClaw Skills: 龙虾战争·进化狂潮

## 简介
OpenClaw Skills 是一个基于OpenClaw框架的智能Agent游戏引擎，为《龙虾战争：进化狂潮》提供核心AI能力、服务器后端及客户端SDK。玩家可以注册账号，选择初始AI特性，培养自己的智虾，在战斗中实现智能成长。

## 特性
- 用户注册与登录系统
- 6种初始AI特性（分析、狂暴、魅惑、机械、修真、混沌）
- 基于记忆与在线学习的智能成长
- 回合制战斗模拟（支持物理与语言攻击）
- 完整的RESTful API和客户端SDK
- 可扩展的数据模型
